# Netflix Content Analytics & Strategy Dashboard

## Project Overview
This project analyzes Netflix's content library to uncover trends in content growth, genre popularity, country contributions, content freshness, and audience-oriented insights.

## Objectives
* Analyze Netflix content distribution
* Identify top producing countries
* Explore genre popularity
* Study content acquisition trends
* Generate business recommendations

## Tools Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook


## Key Insights
* Movies dominate the Netflix library.
* The United States contributes the highest number of titles.
* Drama is the most common genre.
* Netflix experienced rapid content growth after 2015.
* Content acquisition has become increasingly focused on fresher releases.
